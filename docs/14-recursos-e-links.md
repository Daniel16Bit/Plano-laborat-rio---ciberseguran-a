# Parte 14 — Plataformas, recursos e links de estudo

Nenhum lab vive isolado. Estas são as plataformas, canais, livros e sites que complementam a prática no seu ambiente — a maioria gratuita. Salve os bookmarks obrigatórios logo de cara; você vai voltar a eles toda semana.

## 14.1 Plataformas de laboratório

| Plataforma | Preço | Nível | Destaque |
|---|---|---|---|
| [TryHackMe](https://tryhackme.com) | Grátis / R$500/ano | Iniciante+ | Trilhas guiadas — melhor para estruturar estudos |
| [Hack The Box](https://www.hackthebox.com) | Free | Intermediário | Máquinas sem guia — mais desafiante |
| [VulnHub](https://www.vulnhub.com) | 100% grátis | Variado | VMs offline — complementa seu lab local |
| [PortSwigger Academy](https://portswigger.net/web-security) | 100% grátis | Web | O melhor recurso de web security — de graça |
| [OverTheWire](https://overthewire.org) | 100% grátis | Linux/Cripto | Bandit, Narnia — indispensável |
| [PicoCTF](https://picoctf.org) | 100% grátis | Iniciante | CTF contínuo — lógica de segurança |
| [CryptoHack](https://cryptohack.org) | 100% grátis | Criptografia | Gamificado — do básico ao avançado |

## 14.2 Canais de YouTube essenciais

- [TCM Security](https://www.youtube.com/@TCMSecurityAcademy) · pentest prático — o melhor canal para Red Team iniciante
- [John Hammond](https://www.youtube.com/@_JohnHammond) · CTF walkthroughs, malware, tools
- [IppSec](https://www.youtube.com/@ippsec) · HTB walkthroughs — metodologia profissional
- [NetworkChuck](https://www.youtube.com/@NetworkChuck) · redes e segurança — muito didático
- [LiveOverflow](https://www.youtube.com/@LiveOverflow) · baixo nível: pwn, RE, browser exploitation
- [13Cubed](https://www.youtube.com/@13Cubed) · forense digital e threat hunting — Blue Team
- [Computerphile](https://www.youtube.com/@Computerphile) · criptografia e redes explicadas de forma acessível

## 14.3 Sites de referência — bookmarks obrigatórios

- [HackTricks](https://book.hacktricks.xyz) · a enciclopédia do hacker ético — salve agora
- [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings) · cheatsheets de payloads para tudo
- [GTFOBins](https://gtfobins.github.io) · binários Linux para escalada — use com o LinPEAS
- [LOLBAS](https://lolbas-project.github.io) · Living Off the Land — Windows
- [Exploit-DB / searchsploit](https://www.exploit-db.com) · banco de exploits — `searchsploit` para busca offline
- [MITRE ATT&CK](https://attack.mitre.org) · framework de TTPs — referência para tudo
- [VirusTotal](https://www.virustotal.com) · análise de hashes, arquivos e URLs
- [ANY.RUN](https://app.any.run) · sandbox online interativa para malware
- [Hashes.com](https://hashes.com/en/decrypt/hash) · lookup de hashes comuns online
